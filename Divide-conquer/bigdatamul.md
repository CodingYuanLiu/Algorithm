## 大整数乘法
* 设a,b为n位整数。求a*b
原来的复杂度：O(n^2)

* 改善复杂度:

若只是简单分治
```
令:
a = w * 2^(n/2) + x
b = y * 2^(n/2) + z 
则：
a*b = wy*2^n + (wz + xy)*2^(n/2) + xz
总共需算四次小乘法
T(n)=4T(n/2) + bn, T(n) = O(n^2)
```

改良:

```
wz + xy = (w+x)(y+z)-wy-xz
故总共只需要算三次小乘法
T(n) = 3T(n/2) + bn
===> T(n) = O(n^log3) = O(n^1.59)
```
